# Music Release Manager

Учебное Flask-приложение для курсового проекта по дисциплине «Разработка веб-приложений».

Тема проекта: веб-приложение для управления музыкальными релизами независимых артистов с личным кабинетом и аналитикой прослушиваний.

## Возможности

- регистрация и вход пользователей;
- роли пользователей: `admin`, `artist`, гость без авторизации;
- личный кабинет артиста;
- редактирование профиля и аватарки;
- CRUD для музыкальных релизов;
- CRUD для треков внутри релиза;
- загрузка обложек релизов;
- загрузка аудиофайлов треков;
- проигрывание загруженных аудиофайлов через HTML-тег `<audio>`;
- управление площадками публикации;
- ввод статистики прослушиваний;
- страница аналитики;
- выгрузка отчёта по релизам в CSV;
- админ-панель для управления пользователями.

## Структура проекта

```text
data/                  # SQLite-база данных создаётся автоматически
static/css/            # стили приложения
static/uploads/        # загруженные обложки, аудио и аватары
templates/             # HTML-шаблоны Jinja2
app.py                 # основной файл Flask-приложения
schema.sql             # схема базы данных
requirements.txt       # зависимости Python
Procfile               # команда запуска для Railway/Heroku
railway.json           # настройки Railway
README.md              # описание проекта
```

## Установка и запуск локально

```bash
python -m venv .venv
```

Windows PowerShell:

```bash
.venv\Scripts\activate
```


Установка зависимостей:

```bash
pip install -r requirements.txt
```

Запуск:

```bash
python app.py
```

После запуска откройте в браузере:

```text
http://127.0.0.1:5000
```

## База данных

База данных SQLite создаётся автоматически в файле:

```text
data/app.db
```

Если нужно пересоздать базу, удалите файл `data/app.db` и снова запустите приложение.
